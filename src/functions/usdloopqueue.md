# $loopQueue

This function loops/unloops the whole queue, and returns whether it's looped. Returns boolean.

### Usage
```php
$loopQueue
```

## Example

```javascript
bot.command({
name: "loopQueue",
code: `Looped the queue! $loopQueue`
})
```
